# tinker Website

Official website for tinker AI - AI-native personalized learning platform.

## Project Structure

```
tinker Website/
├── src/                    # Source assets
│   └── assets/            # Images, fonts, and media
├── public/                # Public assets
├── docs/                  # Documentation files
├── custom_font/           # Custom font files (Baloo, Montserrat)
├── index.html             # Home page
├── company.html           # Company information page
├── contact.html           # Contact page
├── pricing.html           # Pricing page
├── careers.html           # Careers page
├── privacy.html           # Privacy policy page
├── terms.html             # Terms of service page
├── script.js              # Main JavaScript file
├── terms-styles.css       # Terms page styles
├── package.json           # Node dependencies
└── README.md              # This file
```

## Pages

- **Home** (`index.html`) - Main landing page
- **Company** (`company.html`) - About tinker
- **Contact** (`contact.html`) - Contact information and form
- **Pricing** (`pricing.html`) - Pricing plans
- **Careers** (`careers.html`) - Career opportunities
- **Privacy** (`privacy.html`) - Privacy policy
- **Terms** (`terms.html`) - Terms of service

## Development

### Prerequisites
- Node.js installed
- npm or yarn

### Installation
```bash
npm install
```

### Running the Development Server
```bash
npm run serve
```

The website will be available at `http://localhost:3000`

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Three.js (for 3D graphics)
- Custom fonts: Baloo 2, Montserrat
